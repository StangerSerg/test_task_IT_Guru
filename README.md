# Order Service — тестовое задание

Сервис для управления заказами и товарами с REST API, PostgreSQL и асинхронным FastAPI.

### Требования
- Docker / Docker Compose
- Python 3.12+ (для локального запуска)

### Запуск через Docker (рекомендуется)

docker-compose up -d

### API Документация
После запуска сервера документация доступна по URL:

Swagger UI (интерактивная): http://localhost:8001/docs
